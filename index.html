Projects

Credit Card Fraud Detection using GANs

Developed an advanced fraud detection system using a highly imbalanced credit card transaction dataset containing more than 284,000 records. The project focused on improving fraud detection accuracy by addressing class imbalance using Generative Adversarial Networks (GANs).

Synthetic fraud samples were generated using GANs and compared with traditional resampling techniques such as SMOTE and ADASYN. Multiple machine learning models including Random Forest were evaluated using Precision, Recall, F1-Score and ROC-AUC metrics to identify the most effective fraud detection approach.
